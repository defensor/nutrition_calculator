# Nutrition Tracker (КБЖУ)

Веб-приложение для учета питания с поддержкой сложных блюд, рецептов и учета уварки/ужарки.

## Функционал

- **Продукты:** Создание и редактирование базы продуктов (БЖУ, Ккал на 100г).
- **Блюда (Рецепты):** Создание сложных блюд из продуктов.
  - Поддержка "веса готового блюда" (для учета ужарки/уварки).
  - Автоматический расчет калорийности готового блюда.
- **Дневник:**
  - Учет съеденного по приемам пищи (Завтрак, Обед, Ужин, Перекус).
  - Поддержка нескольких пользователей.
  - Логика "Съел все блюдо" или ввод веса порции.

## Установка и Запуск

### Предварительные требования
- Python 3.8+
- Node.js 14+

### 1. Backend (API)

```bash
cd backend
python3 -m venv venv
source venv/bin/activate  # На Windows: venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload
```

Сервер будет доступен по адресу: `http://localhost:8000`
Swagger документация: `http://localhost:8000/docs`

### 2. Frontend (Интерфейс)

```bash
cd frontend
npm install
npm run dev
```

Приложение будет доступно по адресу: `http://localhost:5173`

## Структура Проекта

- `backend/`: FastAPI приложение, SQLite база данных (`sql_app.db` создается автоматически).
- `frontend/`: React приложение (Vite + Tailwind CSS).

## Использование

1. Откройте приложение в браузере.
2. Создайте пользователя (кнопка `+` рядом с выбором пользователя).
3. Добавьте базовые продукты во вкладке "Products".
4. Создайте блюдо во вкладке "Dishes", добавив ингредиенты. Если вес готового блюда отличается от суммы сырых (ужарка), снимите галочку "Auto" и введите вес.
5. Записывайте съеденное в "Diary".
